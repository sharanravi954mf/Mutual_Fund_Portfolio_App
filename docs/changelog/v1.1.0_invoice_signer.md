# Changelog: Release v1.1.0 (Invoice Signer & Ingestion Upgrades)

## [1.1.0] - 2026-07-19

### Added
* Standalone **Invoice Signer** tab inside the Admin Dashboard.
* In-memory batch ZIP file decompression, signing, and re-zipping inside the `sign-stamp-invoice` edge function.
* Conditional compile-safe cross-platform file picking helpers in `lib/utils/file_picker_helper.dart`.
* Cache-busting service worker unregistration script inside `index.html`.

### Changed
* Optimized cams ingestion loops using SQL batched RPC stored procedures.
* Standardized decimal precision to 6 decimal places on CAMs record imports.
